Чтобы скачать с гитхаба проект:
	Нажмите кнопку "Clone or download" и выберете подходящий вариант (например скачать ZIP архив).
	
После скачки разархивируйте архив в удобную вам папку.

Заходим в папку:
Папка...
В файле:
Путь к файлу...
Указываем/заполняем то-то и то-то. А в файле:
Путь к файлу...
Делаем вот это. Далее запускаем команду:
Команда...
Ждём когда запуститься сервер. После запуска сервера заходим в браузере по адресу:
http://locahost:и остальное содержание адреса...
И видим результат.
